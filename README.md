# hetty.xyz

Source code of [https://hetty.xyz](https://hetty.xyz).

## Installation

```
$ yarn
```

## Local Development

```
$ yarn start
```

This command starts a local development server and opens up a browser window.
Most changes are reflected live without having to restart the server.

## Build

```
$ yarn build
```
